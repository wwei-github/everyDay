# 35. 搜索插入位置

- 二分法模版套入
  
```js
var searchInsert = function (nums, target) {
    let left = 0,right = nums.length;
    let mid
    while(left<right){
         mid = Math.floor((left+right)/2);
        if(nums[mid]===target){
            return mid
        }else if(nums[mid]<target){
            left = mid+1
        }else{
            right = mid
        }
    }
    return left
};
```
